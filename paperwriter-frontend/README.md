# PaperWriter Frontend

AI 驱动的学术论文写作编辑器前端应用。

## 技术栈

- React - UI 框架
- Vite - 构建工具
- TypeScript - 类型安全
- Zustand - 状态管理
- Tailwind CSS - 样式

## 项目结构

```
src/
├── components/      # 组件库
│   ├── layout/      # 布局组件
│   ├── project/     # 项目组件
│   ├── files/       # 文件组件
│   ├── editor/      # 编辑器组件
│   └── ai/          # AI 组件
├── pages/           # 页面
├── services/        # API 服务
├── store/           # 状态管理
├── types/           # 类型定义
└── styles/          # 样式文件
```

## 运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
